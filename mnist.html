MNIST Computer Vision Project

This project demonstrates the implementation of a Convolutional Neural Network (CNN) for image classification using the MNIST dataset. Below is an overview of the steps taken to prepare and set up the project:

  1. Preprocessed the MNIST dataset to prepare it for training.
  2. Designed and trained a CNN model using TensorFlow.
  3. Evaluated the model's performance and fine-tuned hyperparameters for optimization.
  4. Deployed the trained model to classify handwritten digits accurately.
