QuickBase Automation Project

This project involved automating data retrieval, manipulation, and transmission using Bash scripting with QuickBase API integration.

The main components and steps involved in this project were:

  • Setting up variables for QuickBase realm, user token, and database IDs.
  • Constructing API requests and headers for interacting with QuickBase.
  • Using cURL commands to make API requests and handle responses.
  • Retrieving data from QuickBase, processing it with Bash scripts, and sending modified data back.

While the project successfully automated certain billing processes, there is still room for optimization. Due to time constraints, a more efficient function was not implemented.
